# CV Matcha

A web-based CV management system where recruiters create position templates from a reusable attribute library, and candidates build profiles and generate tailored CVs with automatic assembly. Supports light/dark themes and two languages.

**Live:** [cv-matcha-lyart.vercel.app](https://cv-matcha-lyart.vercel.app)

## Other dependencies

- **Better Auth** – authentication (email/password, Discord, GitHub)
- **Cloudinary** – image upload and storage
- **jsPDF** – PDF CV generation
- **Mantine Dates** – date picker components
- **Sonner** – toast notifications
